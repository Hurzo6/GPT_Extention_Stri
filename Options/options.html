Comment utiliser cette extension :

1. Sélectionnez du texte sur n'importe quelle page Web.

2. Cliquez avec le bouton droit pour afficher le menu contextuel.

3. Choisissez "Répondre" pour obtenir une réponse de l'assistant.

Comment obtenir une clé API :

1. Allez sur le site Web d'OpenAI et créez un compte.

2. Accédez à la section API pour générer une nouvelle clé.

3. Copiez cette clé et collez-la dans le champ API Key des options de cette extension.

⚠️ Attention : Si vous souhaitez utiliser votre propre clé API, vous devez au minimum avoir crédité votre compte de 5€. Sinon, la clé sera invalide car il ne pourra pas vous facturer pour les requêtes.
